Introduction to Python

Below find a few basic facts when entering the world of Python

Python is....

Python is an interactive, object-oriented programming language. It is used for several different applications including web development, software development, mathematic computations, and system scripting.

It was developed and given its name by Guido van Rossum who at the time was reading Monty Pythonâ€™s Flying Circus.

Python because....

Python has grown in popularity because it's relatively easy to learn and contains a vast standard library. It's syntax is similar to the English language and allows for programs to be written in fewer lines of code.

Additionally Python is compatible with different platforms such as Windows, Mac, Linux, and Raspberry Pi. Because python utilizes interactive editors and interpreters, it can be tested while programming.

One of the key benefits of this programming language is the ability to automate almost anything using it's built-in modules, pre-written code, or custom scripts. It's wide adoption can also be attributed to the fact that it is free and open source.
